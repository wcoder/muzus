# muzus
New conception of audio player for web. Based on [playeros](https://github.com/wcoder/playeros).

## Development

### Gulp

Install `gulp-cli`:
```
npm install --global gulp-cli
```

Restore development dependencies:
```
npm i --only=dev
```

### SCSS

Compiling:
```
gulp sass
```

Auto-compiling:
```
gulp sass:watch
```
